
## Installing Steps

Install EmacsMac@30 from https://github.com/jdtsmith/emacs-mac

```sh
git clone ...
```

## Features

- Common Lisp IDE (SLIME Integration)
- Nerd Style
- Good with AI Coding

## How to install?

> ⚠️: This will overwrite all files in `~/.emacs.d/init.el` and `~/.emacs.d/hikettei`. Proceed with caution!

```
$ git clone git@github.com:hikettei/init.el.git
$ cd ./init.el
$ make install
```
